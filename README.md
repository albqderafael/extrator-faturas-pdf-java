# extrator-faturas-pdf-java
# Extrator de Faturas PDF em Java

## Descrição

Projeto desenvolvido em Java com o objetivo de realizar a **extração e processamento de dados de faturas em formato PDF**, convertendo informações não estruturadas em dados organizados e manipuláveis.

A aplicação lê arquivos PDF, extrai os dados relevantes e realiza o **parse das informações**, transformando-as em objetos Java para posterior uso e análise.

---

## Funcionalidades

* Leitura de arquivos PDF
* Extração de dados textuais
* Parse e estruturação das informações
* Conversão dos dados em objetos Java
* Organização do código em camadas (main, util, modelos)

---

## Tecnologias Utilizadas

* Java
* Manipulação de arquivos (java.io)
* Processamento de dados
* Programação Orientada a Objetos (POO)

---

## Como o projeto funciona

1. O sistema realiza a leitura de um arquivo PDF contendo dados de fatura
2. O conteúdo textual é extraído do documento
3. Os dados são processados e interpretados (parse)
4. As informações são organizadas em classes Java (modelos)
5. O resultado final pode ser utilizado para análise, armazenamento ou integração com outros sistemas

---

## Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/extrator-faturas-pdf-java.git
```

2. Abra o projeto em uma IDE (IntelliJ recomendado)

3. Execute a classe:

```
FaturaMain.java
```
